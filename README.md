# Rebound-Chrome-Extension
Read breaking news using chrome extension on a click


## Built With

* Javascript
* JQuery
* Materializecss
* News-api

## Installation

Windows & Linux:

```sh
Open google chrome (or Opera/firefox)
```

```sh
Go to More tools -> extensions
```

```sh
add unpacked extensions
```

```sh
start using :)
```
